# 🤖 ia-bot-ahmed

AI-powered coding assistant — fork of [opencode](https://github.com/anomalyco/opencode).

### Quick Install

```bash
git clone https://github.com/Ahmedmecatronique/ia-bot_ahmed.git
cd ia-bot_ahmed
bun install
bun dev
```

### Prerequisites

- [Bun](https://bun.sh) — `powershell -c "irm bun.sh/install.ps1 | iex"`

### Usage

```bash
bun dev                    # Launch TUI
ia-bot-ahmed --help        # CLI help
```
