ชุดโครงการชุมชนที่สร้างขึ้นบน ia-bot-ahmed

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ ia-bot-ahmed ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-ia-bot-ahmed](https://github.com/awesome-ia-bot-ahmed/awesome-ia-bot-ahmed) และ [ia-bot-ahmed.cafe](https://ia-bot-ahmed.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
