ia-bot-ahmed kan kjøres som en nettapplikasjon i nettleseren din, og gir den samme kraftige AI-kodingsopplevelsen uten at du trenger en terminal.

![ia-bot-ahmed Web - Ny økt](../../../assets/web/web-homepage-new-session.png)

## Komme i gang

Start web-grensesnittet ved å kjøre:

```bash
ia-bot-ahmed web
```

Dette starter en lokal server på `127.0.0.1` med en tilfeldig tilgjengelig port og åpner automatisk ia-bot-ahmed i standard nettleser.

---

## Konfigurasjon

Du kan konfigurere webserveren ved å bruke kommandolinjeflagg eller i [konfigurasjonsfilen](/docs/config).

### Port

Som standard velger ia-bot-ahmed en tilgjengelig port. Du kan spesifisere en port:

```bash
ia-bot-ahmed web --port 4096
```

### Vertsnavn

Som standard binder serveren seg til `127.0.0.1` (kun localhost). Slik gjør du ia-bot-ahmed tilgjengelig på nettverket ditt:

```bash
ia-bot-ahmed web --hostname 0.0.0.0
```

---

### Autentisering

For å beskytte tilgang, angi et passord ved hjelp av miljøvariabelen `IA_BOT_AHMED_SERVER_PASSWORD`:

```bash
IA_BOT_AHMED_SERVER_PASSWORD=secret ia-bot-ahmed web
```

Brukernavnet er satt til `ia-bot-ahmed` som standard, men kan endres med `IA_BOT_AHMED_SERVER_USERNAME`.

## Bruke webgrensesnittet

Når den er startet, gir nettgrensesnittet tilgang til dine ia-bot-ahmed-økter.

### Sesjoner

Se og administrer øktene dine fra hjemmesiden. Du kan se aktive økter og starte nye.

![ia-bot-ahmed Web - Active Session](../../../assets/web/web-homepage-active-session.png)

### Serverstatus

Klikk på "Se servere" for å se tilkoblede servere og deres status.

![ia-bot-ahmed Web - Se servere](../../../assets/web/web-homepage-see-servers.png)

---

## Koble til en terminal

Du kan koble en terminal TUI til en kjørende webserver:

```bash
# Start the web server
ia-bot-ahmed web --port 4096

# In another terminal, attach the TUI
ia-bot-ahmed attach http://localhost:4096
```

Dette lar deg bruke både nettgrensesnittet og terminalen samtidig, og deler samme økter og tilstand.

---

## Konfigurasjonsfil

Du kan også konfigurere serverinnstillinger i `ia-bot-ahmed.json` konfigurasjonsfilen:

```json
{
  "server": {
    "port": 4096,
    "hostname": "0.0.0.0",
    "mdns": true,
    "cors": ["https://example.com"]
  }
}
```

Kommandolinjeflagg har forrang over konfigurasjonsfilinnstillinger.
